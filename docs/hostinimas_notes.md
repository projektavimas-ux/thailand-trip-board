# Hostinimas ir Supabase Schema

1. Projektas naudoja GitHub Pages (veikia per `docs/` arba repozitorijos root). Front-end kraunamas per statinius HTML.
2. `supabase-config.js` ir `.v2.js` skirti dinaminiam `stage_windows` užkrovimui iš Supabase (skaitymas).
3. `data/*.json` veikia kaip lokali failų DB atsarginė versija.
